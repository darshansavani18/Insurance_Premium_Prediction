# Insurance Premium Prediction

## Project Overview

This project performs exploratory data analysis (EDA), data preprocessing, feature engineering, and statistical analysis on an insurance dataset. The objective is to understand the factors that influence medical insurance charges and prepare the data for machine learning models.

---

## Dataset Features

- Age
- Sex
- BMI
- Children
- Smoker
- Region
- Charges

---

## Project Workflow

### 1. Data Exploration
- Dataset inspection
- Missing value checking
- Duplicate value checking
- Statistical summary

### 2. Exploratory Data Analysis (EDA)
- Distribution plots
- Count plots
- Box plots
- Correlation heatmap

### 3. Data Preprocessing
- Label Encoding
- One-Hot Encoding
- Feature Scaling

### 4. Feature Engineering
- BMI category creation
- Data transformation

### 5. Statistical Analysis
- Pearson Correlation Test
- Chi-Square Test
